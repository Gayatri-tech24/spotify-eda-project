# Spotify Songs EDA Project

## Overview
This project performs Exploratory Data Analysis (EDA) on a Spotify songs dataset using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Top popular songs
- Popularity distribution
- Danceability vs Popularity
- Correlation heatmap
- Artist analysis

## Key Insights
- Most songs have low popularity scores.
- Popularity distribution is highly skewed.
- Danceability alone does not determine popularity.
- Energy and loudness are positively correlated.

## Dataset Source
Spotify Tracks Dataset from Kaggle:
https://www.kaggle.com/datasets/maharshipandya/-spotify-tracks-dataset
